This real-time video translator app is built using Flask, a web framework in Python. The application utilizes the Google Translate API for language translation. It includes two main components - a Flask web server (app.py) and a desktop application (desktop_app.py). The server serves as an interface for users to access the translation service.

This real-time video translator application defaults to using the microphone for user input, allowing users to speak or input text for translation. Users can speak in their preferred language, and the application translates the spoken words into the desired target language.

For those who require translatation for an audio output, the application facilitates this by recommending the setup of the stereo mixer(in Windows). The stereo mixer enables the capture of system audio, providing users with a seamless experience of translating the content they view may it be video calls or any other media content.

TO USE STEREO MIXER WITH THIS PROJECT:
Make sure stereo mixer is enabled.
Select stereo mixer as the input for the system and speaker as the output
Run any background audio and execute the program to translate
